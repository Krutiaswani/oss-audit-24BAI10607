# Steps to Run the Scripts on Linux

1. Open Terminal

2. Navigate to the folder containing scripts:

   ```bash
   cd path/to/your/folder
   ```

3. Give execution permission:

   ```bash
   chmod +x script1.sh
   ```

4. Run the script:

   ```bash
   ./script1.sh
   ```

5. Repeat the same steps for other scripts:

   * script2.sh
   * script3.sh
   * script4.sh
